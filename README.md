# Transaction list

##### Небольшое Angular приложение.

<!-- ### [Демо](https://alenaismagilova.github.io/test-task-only/) -->

### Стек

1. Angular 19, TypeScript 5
2. Rxjs 7
3. NgRx
4. scss

### Быстрый старт

1. Клонирование и установка зависимостей

```markdown
git clone https://github.com/AlenaIsmagilova/transaction-list.git
```

```markdown
cd transaction-list
```

```markdown
ng build
```

2. Запуск проекта

```markdown
ng serve
```

3. Откройте в браузере http://localhost:4200
